# Skills registry

Discover skills here during the boot sequence. If `NEXT.md` matches a skill below,
load it and follow its checklist. If nothing matches, say "none found" and proceed
mechanically — never invent a skill.

> Author a new skill by copying `_template/SKILL.md` into `skills/<name>/SKILL.md`
> and adding a row here.

| Skill | When to use | Path |
|---|---|---|
